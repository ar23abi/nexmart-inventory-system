# Nexmart – Multi-Branch Inventory Management System

## Overview
Nexmart is a full-stack web application developed to manage inventory across multiple retail branches. It allows customers to browse and reserve products, while administrators and warehouse staff manage stock and transfer inventory between locations.

## Features
- Customer product browsing and reservation
- User registration system
- Store selection (Hatfield, Watford, London)
- Barcode-based reservation confirmation
- Admin dashboard for product and stock management
- Warehouse module for stock handling
- Stock transfer between branches

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## System Modules

### Customer
- Browse products
- Select store
- Reserve items

### Admin
- Manage products and stock
- Monitor system

### Warehouse
- Manage inventory
- Transfer stock between branches

## How to Run
1. Install XAMPP
2. Move project folder to htdocs
3. Import database into phpMyAdmin
4. Run on localhost
